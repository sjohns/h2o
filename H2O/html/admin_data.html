H2O Packing Data Admin — Manage and publish packing data

Workflow

  1. Download — Get the current data as an Excel workbook. Open it in Excel or Google Sheets.
  2. Edit — Add, remove or update SKUs. Use the dropdown in the Active column (Y/N). Do not add or remove columns.
  3. Upload — Select your modified file using the file picker below.
  4. Validate — Check the file for errors before continuing.
  5. Preview — Review exactly what will change compared to the live data.
  6. Change reason — Briefly describe what changed and why.
  7. Publish — Push changes live. The packing tool updates immediately.

Tips

  • The Active column only accepts Y or N. Use the dropdown in Excel.
  • Display Order, Length, Popularity Score, and bundle sizes must be positive whole numbers.
  • The Product Type column on the SKUs sheet must match a name on the Product Types sheet.
  • Changes take effect immediately upon publish — the packing tool updates live.
  • All publishes are versioned. Use Publish History below to review past changes.
  • Download load_packing_data.js to get the data file used by the original packing system.
