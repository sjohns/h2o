# Solver Harness

Inputs:

- `orders.json`: local batch test orders
- `expected_results.json`: expected outputs for local batch orders
- `orders_historical.json`: historical orders converted from `python_process/cloud_app/tests/parity_cases.json`
- `expected_historical.json`: historical expected outputs captured from the original JS solver

Run local harness:

```bash
cd H2O
python3 api/tests/harness/run_harness.py
```

Run historical harness:

```bash
cd H2O
python3 api/tests/harness/run_harness.py --historical
```

Regenerate local expected outputs (current solver only):

```bash
cd H2O
python3 api/tests/harness/run_harness.py --regenerate
```

Historical expected outputs are fixed and are not regenerated by `run_harness.py`.

One-time converter used to create historical files:

```bash
cd H2O
python3 api/tests/harness/convert_historical_from_parity_cases.py
```
